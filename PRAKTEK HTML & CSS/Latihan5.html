Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lain, yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mangatur warna latar belakang, gaya huruf, jarak anatar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML melainkan sebagai pelengkap agar tampilan dokumen HTML lebih indah dan dinamis. Berikut adalah contoh dokumen CSS

Bagaimana Menggunakan CSS?

Ada 3 cara menggunakan CSS:

  • Inline Style : Menuliskan kode CSS langsung pada tag yang akan diubah atau diatur style-nya
  • Internal/Embedded Style : Menuliskan kode CSS pada bagian header dokumen HTML
  • Eksternal/Linked Style : Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag dari dokumen HTML
